# lib/helpers — ヘルパー・ユーティリティ

アプリの各層から横断的に使われる補助ロジック群です。
認証・アクセス制御・バリデーション・Firebase診断・UI補助など、単一責務の静的ヘルパーやサービスクラスで構成されています。

---

## ファイル一覧

### 認証系 (Auth)

| ファイル名               | 主なクラス / 関数                                                 | 役割                                                                                                                                                              |
| ------------------------ | ----------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `auth_service.dart`      | `AuthService`                                                     | Firebase Auth のラッパーインターフェース。全メソッドが `UnimplementedError` をスローするスタブで、本番コードでは `MockAuthService` か実 Firebase を使うことを促す |
| `auth_state_helper.dart` | `AuthStateHelper`                                                 | 現在の Firebase `User` を検査し、QR 招待など要認証機能をゲートする。未認証時に表示する「サインアップ必須」ロック画面ウィジェットも提供                            |
| `mock_auth_service.dart` | `MockUser`, `MockUserCredential`, `MockAuthService`               | `dev` フレーバー向けの Firebase Auth モック。メールアドレスのプレフィックスからダミー表示名・UID を生成し、実 Firebase 資格情報なしで認証フローをテストできる     |
| `firebase_stub.dart`     | `FirebaseAuth`, `User`, `UserCredential`, `FirebaseAuthException` | `dev` フレーバー向けに Firebase Auth の API 外形を模倣する軽量スタブ。全メソッドは `UnimplementedError` をスロー                                                  |

---

### アクセス制御・バリデーション系 (Access Control & Validation)

| ファイル名                   | 主なクラス / 関数                       | 役割                                                                                                                                                     |
| ---------------------------- | --------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `access_control_helper.dart` | `AccessControlHelper`                   | グループへの参加確認・リストの閲覧/編集権限・招待権限などを静的メソッドで提供するランタイム権限チェック層                                                |
| `security_validator.dart`    | `SecurityValidator`                     | Firebase Auth UID を `SharedGroup` のメンバーリスト・オーナーフィールド・招待権限に照合し厳格に検証する。`dev` フレーバーでは全チェックをスキップ        |
| `validation_service.dart`    | `ValidationService`, `ValidationResult` | グループ名（空・文字数・重複）とメンバーメールアドレス（フォーマット・重複）の入力バリデーションを一元管理。例外ではなく型付き `ValidationResult` を返す |

---

### Firebase 診断系 (Firebase Diagnostics)

| ファイル名                  | 主なクラス / 関数     | 役割                                                                                                                                                   |
| --------------------------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `firebase_diagnostics.dart` | `FirebaseDiagnostics` | Firebase 接続の健全性チェックスイート（Auth 状態・Firestore 読み取りレイテンシ・認証済み書き込みテスト）を実行し、結果を `Map<String, dynamic>` で返す |

---

### UI・ユーティリティ系 (UI & Utils)

| ファイル名                   | 主なクラス / 関数    | 役割                                                                                                                                                                                          |
| ---------------------------- | -------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `ui_helper.dart`             | `UiHelper`           | SnackBar の表示ロジックを集約する。成功（緑）・エラー（赤）・警告（オレンジ）・情報の静的メソッドを提供し、各メソッドに `context.mounted` ガードを内包                                        |
| `misc.dart`                  | `logger` (`Logger`)  | `logger` パッケージの共有インスタンスをグローバルに公開する 1 行ファイル。ヘルパー群全体で共通のロギングユーティリティとして import される                                                    |
| `dev_utils_helper.dart`      | `DevUtilsHelper`     | `dev` フレーバー限定のデバッグ補助。確認ダイアログを経て全 Hive データをクリアし関連 Riverpod プロバイダーをリセットするボタンウィジェットを提供（`prod` では非表示）                         |
| `qr_code_helper.dart`        | `QrCodeHelper`       | QR スキャン処理と招待ペンディング確認を `AuthStateHelper` のチェックでゲート。未認証ユーザーにはサインアップ促進ダイアログを表示し、QR スキャンボタンウィジェットのビルダーも提供             |
| `user_id_change_helper.dart` | `UserIdChangeHelper` | サインイン後のユーザーコンテキスト切り替えを統括。静的な `_activeTransition` Future で二重起動を防ぎながら、Hive 再初期化・Firestore 同期・プロバイダー無効化・データ移行ダイアログを順次実行 |
