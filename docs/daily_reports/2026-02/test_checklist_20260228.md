## 2. ホワイトボード機能テスト

### アクセス方法

- [❌] グループメンバー管理ページでホワイトボードプレビューが表示される←表示されていない。
- [☑] プレビューをダブルタップしてエディターを開ける
- [☑] グループ共有ホワイトボードを開ける
- [☑] 個人用ホワイトボードを開ける（ダブルタップまたはメニュー）
- [☑] 他メンバーのホワイトボードを閲覧できる
- [☑] グループページ横長表示で、左側のプレビューから直接エディター開放

### 描画機能

- [☑] 6色カラーピッカーが表示される（黒、赤、緑、黄、カスタム2色）
- [☑] 選択した色で描画できる
- [☑] ペン太さが3段階で選択できる
- [☑] 複数のストロークを描画できる（自動分割）
- [☑] 複数色のストロークが混在して描画可能
- [☑] ストロークが30px以上離れている場合は自動分割される

### ツールバー機能（2段構成）

- [☑] 上段：6色カラーボタン + スクロール/描画モード切り替えアイコン
- [☑] 下段：3段階ペン太さボタン + ズーム±ボタン + 全消去ボタン
- [☑] 全ツールバーアイコンが縦横両方向で表示される

### スクロール/描画モード

- [☑] スクロールロックボタンが動作する
- [☑] 🖌️描画モード（ロック）中：描画可能、スクロール不可
- [☑] 🖱️スクロールモード（開放）中：スクロール可能、描画不可
- [☑] モード表示が視覚的に分かりやすい

### ズーム機能

- [☑] ズームアウトボタン（-）で0.5x縮小
- [☑] ズームインボタン（+）で0.5x拡大
- [☑] 現在のズーム倍率が表示される（例：1.0x, 1.5x, 2.0x）
- [☑] ズーム時にスクロール範囲が自動調整される
- [☑] 拡大時にスクロール可能な範囲が増加する

### 保存機能

- [ ] 「保存」ボタンで保存できる
- [ ] 保存完了メッセージが表示される
- [ ] 再度開いた時に描画内容が復元される
- [ ] Firestore + Hive 両方に同期される
      グループ共通ボードが保存されてないようです

### 全消去機能

- [☑] 「全消去」ボタン（ゴミ箱アイコン）が常時表示される
- [☑] 確認ダイアログが表示される
- [☑] すべての描画が消去される

### 閲覧専用モード

- [☑] 他メンバーのホワイトボードは(編集禁止スイッチオンの時は)「閲覧専用」バーが表示される
- [☑] ツールバーが表示されない
- [☑] 描画機能が無効化されている

## 使用方法

### 新しいチェックリスト作成時

1. このテンプレートをコピー
2. ファイル名: `test_checklist_YYYYMMDD.md`
3. ヘッダー情報を記入（テスト日、担当者、バージョン等）
4. 「🔥 重点テスト項目」セクションに今回の実装内容を記載
5. 該当しないテスト項目は削除または調整
6. テスト結果サマリーの項目数を更新

### テスト実施時

1. 🔥 重点テスト項目から開始（最優先）
2. 基本機能テストを実施
3. 各セクションのチェックボックスをチェック
4. 問題が見つかったら「検出された問題」セクションに記録
5. テスト結果サマリーを記入
6. 総合判定を記入
