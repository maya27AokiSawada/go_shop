## 2. ホワイトボード機能テスト

### アクセス方法

- [☒] グループメンバー管理ページでホワイトボードプレビューが表示される
- [ ] プレビューをタップしてエディターを開ける　←ダブルタップになってるけどタップにする？個人用ホワイトボードがダブルタップだから合わせるためにこのままでいいと思う
- [☑] グループ共有ホワイトボードを開ける
- [☑] 個人用ホワイトボードを開ける（ダブルタップまたはメニュー）メニュー機能はないです。
- [☑] 他メンバーのホワイトボードを閲覧できる
- [☑] グループページ横長表示で、左側のプレビューから直接エディター開放

### 描画機能

- [☑] 6色カラーピッカーが表示される（黒、赤、緑、黄、カスタム2色）
- [☑] 選択した色で描画できる
- [☑] ペン太さが3段階で選択できる
- [☑] 複数のストロークを描画できる（自動分割）
- [☑] 複数色のストロークが混在して描画可能
- [☑] ストロークが30px以上離れている場合は自動分割される

### ツールバー機能（2段構成）

- [☑] 上段：6色カラーボタン + スクロール/描画モード切り替えアイコン
- [☑] 下段：3段階ペン太さボタン + ズーム±ボタン + 全消去ボタン
- [☑] 全ツールバーアイコンが縦横両方向で表示される

### スクロール/描画モード

- [☑] スクロールロックボタンが動作する
- [☑] 🖌️描画モード（ロック）中：描画可能、スクロール不可
- [☑] 🖱️スクロールモード（開放）中：スクロール可能、描画不可
- [☑] モード表示が視覚的に分かりやすい

### ズーム機能

- [☑] ズームアウトボタン（-）で0.5x縮小
- [☑] ズームインボタン（+）で0.5x拡大
- [☑] 現在のズーム倍率が表示される（例：1.0x, 1.5x, 2.0x）
- [☑] ズーム時にスクロール範囲が自動調整される
- [☑] 拡大時にスクロール可能な範囲が増加する

### 保存機能

- [☑] 「保存」ボタンで保存できる
- [☑] 保存完了メッセージが表示される
- [☑] 再度開いた時に描画内容が復元される
- [ ] Firestore + Hive 両方に同期される　ここはFirestoreへの保存は確認しました

### 全消去機能

- [☑] 「全消去」ボタン（ゴミ箱アイコン）が常時表示される
- [☑] 確認ダイアログが表示される
- [☑] すべての描画が消去される

### 閲覧専用モード

- [ ] 他メンバーのホワイトボードは「閲覧専用」バーが表示される
- [ ] ツールバーが表示されない
- [ ] 描画機能が無効化されている
