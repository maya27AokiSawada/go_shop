# GoShopping 指示書一覧

このディレクトリは、GoShopping プロジェクトの **AI / 開発者向け指示書**を
「プロジェクト共通ルール」と「機能別」に分割して管理します。

## 読み方（重要）

1. **必ず最初に `00_project_common.md` を読む**
2. 次に、作業対象の機能に対応する指示書を読む
3. 実装・修正前にアンチパターンを再確認する

## ファイル一覧

- `00_project_common.md`
  → 全機能共通のアーキテクチャ・禁止事項・必須ルール・頻出アンチパターン

- `20_groups_lists_items.md`
  → グループ・リスト・アイテム管理（Firestore + Hive 差分同期）

- `30_whiteboard.md`
  → ホワイトボード機能（描画・編集ロック・同期・プラットフォーム差）

- `40_qr_and_notifications.md`
  → QR 招待（v3.1）・通知・通知履歴

- `50_user_and_settings.md`
  → 認証・ユーザー管理・設定・アカウント削除

- `90_testing_and_ci.md`
  → テスト戦略・CI/CD・品質保証

---

⚠️ **新規実装・修正時は必ず共通指示書を先に確認すること**
